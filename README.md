# 숭실대학교 대학가 주거 환경 분석 (SSU Housing Analysis)

본 프로젝트는 숭실대학교 인근 대학가(동작구 상도동 및 인접 생활권)의 주거용 부동산(원룸, 오피스텔, 빌라 등) 전월세 데이터를 수집, 검증 및 분석하여 대학생들의 주거 환경 현황과 임대료 추이를 파악하는 것을 목표로 합니다.

---

## 📂 프로젝트 디렉토리 구조 및 규칙

프로젝트의 유지보수성과 확장성을 위해 아래와 같은 구조로 파일 및 폴더를 관리합니다.

```text
ssu-housing-analysis/
├── docs/                # 프로젝트 관련 문서 및 명세서
│   └── project_specification.md  # 데이터 수집 및 검증 단계 명세서
├── srcs/                # 분석 및 데이터 처리를 위한 파이썬 소스 코드
├── data/                # 데이터 저장 공간 (Git 관리 대상 제외 추천)
│   ├── raw/             # 공공데이터포털 API에서 수집한 원시 데이터 (XML/JSON)
│   └── processed/       # 검증 및 정제 파이프라인을 통과한 데이터셋
└── README.md            # 프로젝트 개요 및 실행 방법 가이드
```

- **`srcs/`**: 모든 파이썬 스크립트, 데이터 수집 및 정제 모듈을 이 폴더 안에 위치시킵니다.
- **`docs/`**: 기획서, 데이터 명세서, 가이드 등 프로젝트 개발 이외의 문서 파일들을 보관합니다.
- **`data/`**: 로컬에 다운로드되거나 생성된 데이터가 위치하는 곳으로, 원천 데이터(`raw/`)와 정제 완료된 데이터(`processed/`)를 구분하여 저장합니다.

## 📊 프로젝트 명세 요약 (데이터 수집 및 검증 단계)

상세한 내용은 [docs/project_specification.md](file:///Users/seungjun/Desktop/projects/ssu-housing-analysis/docs/project_specification.md) 문서에서 확인할 수 있습니다.

### 1. 데이터 수집 출처
국토교통부 실거래가 오픈 API (공공데이터포털)를 통해 아래 세 가지 매물 정보를 연계 수집합니다:
1. **단독/다가구 전월세 실거래자료** (원룸, 단독주택)
2. **오피스텔 전월세 실거래자료** (오피스텔)
3. **연립다세대 전월세 실거래자료** (빌라, 다세대주택)

### 2. 수집 및 필터링 기준
- **공간 범위**: 숭실대학교 반경 도보 통학 및 인접 생활권 법정동
  - 동작구 상도동 (`1159010200`), 상도1동 (`1159010300`), 사당동 (`1159010700`)
  - 관악구 봉천동 (`1162010100`)
- **물리 범위**: 전용면적 `40㎡` 이하 (소형 평수 매물만 추출)
- **거래 유형**: **전세** 및 **월세** 거래 정보 (매매 제외)
- **시간 범위**: `2021년 1월 1일 ~ 현재` (최근 약 5개년 시계열 데이터)

### 3. 데이터 검증 파이프라인
수집된 Raw Data의 무결성을 확보하기 위해 다음 4단계 검증 프로세스를 수행합니다.

1. **스키마 및 타입 검증**: 수치 데이터(보증금액, 월세금액, 면적 등)의 타입 파싱 및 스키마 검증
2. **필수 필드 결측치 검증**: 보증금액, 월세금액, 전용면적, 법정동, 계약년월 중 결측치가 있는 레코드 감지 및 배제
3. **논리적 일관성 검증**: 금액 및 면적의 양수 조건, `건축년도 <= 계약년도` 검증, 반지하층 이하 층수 표기 통일
4. **중복 데이터 제거**: 동일 매물/날짜/금액의 중복 수집 데이터(100% 중복)를 제거하고 최신 1건만 유지

## 🛠️ 개발 환경 구축 및 실행 방법

본 프로젝트는 Windows 및 macOS 환경 모두에서 호환되는 실행 환경을 지원합니다.

### 1. Conda 가상환경 생성 및 활성화
안정적인 패키지 관리를 위해 Conda를 활용하여 `Python 3.11` 환경을 구축합니다.

```bash
# 1) python 3.11 기반의 가상환경 생성 (최초 1회 실행)
conda create -n ssu-housing python=3.11 -y

# 2) 가상환경 활성화
conda activate ssu-housing
```

### 2. 필수 라이브러리 설치
프로젝트 루트 디렉토리에서 아래 명령어를 실행하여 필요한 패키지들을 설치합니다.

```bash
pip install -r requirements.txt
```

### 3. API 키 설정 (로컬 환경 변수)
공공데이터포털에서 발급받은 API 인증키를 안전하게 관리하기 위해 로컬 환경 변수 설정을 진행합니다.

```bash
# .env.template 파일을 기반으로 로컬용 .env 파일 생성
cp .env.template .env
```
생성된 `.env` 파일 내부의 `SERVICE_KEY` 항목에 발급받은 실거래가 API 인증키(디코딩된 키 권장)를 입력해 줍니다. 
*(참고: `.env` 파일은 [.gitignore](file:///Users/seungjun/Desktop/projects/ssu-housing-analysis/.gitignore)에 등록되어 있으므로 Git 저장소에 커밋되지 않습니다.)*

### 4. 프로그램 실행 테스트
모든 설정이 완료되었다면 아래 명령어를 수행하여 기본 뼈대 코드를 실행합니다.

```bash
python srcs/main.py
```
**정상 실행 시 출력 예시**:
```text
숭실대학교 대학가 주거 환경 분석 프로젝트 (SSU Housing Analysis) 실행 완료
```
